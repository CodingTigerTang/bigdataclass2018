3.2 View record level data

*Important tips to record preview data**

  1. How many flights on 7/18 were 1 or more hours late?
flights %>%
  filter(depdelay >= 60,
         month == 7,
         dayofmonth == 18) %>%
  tally() 
## # Source:   lazy query [?? x 1]
## # Database: PostgreSQL 9.3.20[rstudio_dev@localhost/postgres]
##                 n
##   <S3: integer64>
## 1            1239
  1. Use filter() to retreive only the needed data, and head() to limit the preview even further.
flights %>%
  filter(depdelay >= 60,
         month == 7,
         dayofmonth == 18) %>%
  head(100)
